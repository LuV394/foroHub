<h3>Endpoints

POST /users: Registrar un nuevo usuario.
POST /login: Iniciar sesión y obtener un token JWT.
POST /topics: Crear un nuevo tópico.
GET /topics: Listar todos los tópicos.
DELETE /topics/{id}: Eliminar un tópico por ID.
